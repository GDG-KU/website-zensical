# 🚀 Fetch란?

**Fetch**는 GDGoC KU에서 매월 1회 진행하는 정기 기술 세미나입니다.

**목적**

- 최신 기술 트렌드 조사 및 분석 공유
- 개발 프로젝트 및 기술 회고 발표
- 커뮤니티 내 지식 공유 및 네트워킹
- 개인 성장과 동시에 커뮤니티 기여 경험 제공

**특징**

- 🌍 **외부인 참석 가능** - 누구나 들을 수 있는 오픈 세미나
- 📹 **전 세션 녹화** - YouTube 계정에 업로드하여 언제든 다시 시청 가능
- 📱 **SNS 콘텐츠 제작** - 각 주제를 카드뉴스로 제작하여 Instagram에 게시

------

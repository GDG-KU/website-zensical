# Point 제도

📌 모든 회원은 **0점**에서 시작합니다.

- `🐤 SWE Junior는`**100점을 넘기고 조건 달성 시 `🐔 SWE Member`로 승급하게 됩니다.**

    > 승급한 구성원의 포인트가 다시 100점 이하로 내려가더라도, 강등되지 않습니다.
    > 

## 공통 항목

- `🐔 Member` 및 🐯`Core` 는 기수 종료 시점에서 **포인트가 100점 이상**이면 수료 가능합니다.
